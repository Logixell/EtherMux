# Command line interface
## Available Commands

Below are the commands supported by the `spe_Cli.c` module:

### help
Displays a list of available commands and their descriptions.
```shell
help
```

### version
Shows the current version of the software.

```shell
version
```

### status
Prints the current system status.

```shell
status
```

### set [parameter] [value]
Sets a configuration parameter to the specified value.

```shell
set <parameter> <value>
```

### get [parameter]
Retrieves the value of a configuration parameter.

```shell
get <parameter>
```

### reset
Resets the system to default settings.

```shell
reset
```



For more details on each command, refer to the source code or use the `help` command in the CLI.
