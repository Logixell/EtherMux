# EtherMux is a protocol designed to run over Single Pair Ethernet (SPE).  
## Status: Architecture/Design Phase  
## Comming Soon: https://EtherMUX.com
Ultra low latency signaling combined with serial data transport.  
- 100 Mbps 802.3bw 100BaseT1 Ethernet phy DP83TC812
- Dual Ethernet phy for dasychain configuration
- Minimal latency Flow through architecture (ie. Not store and forward)
- Custom FPGA Hardware MAC implementing the EtherMux Protocol
- Open source protocol
- Open source Raspberry PI Pico controller code
- Free FPGA configuration file embedded in Pico codebase
