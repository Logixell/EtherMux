# EtherMux Code Base
### Supported Boards:
B2601-R01 EtherMux Hub
### Description
This repository contains the source code for EtherMux Hub configuration software.  It is targeted to run on a Raspberry PI RP2350 (Pico 2) microcontroller.  It has the following functions:  
- Command line interface (USB or Serial)
- Program and Configure FPGA
- Configure Single Pair Ethernet PHY's
- Configure network connectivity
